## 🚀 如何开始

1.  使用 **Cocos Creator** (建议 3.x 版本) 打开本项目文件夹。
2.  打开 `assets/Scene/scene.scene` 场景。
3.  点击编辑器上方的“运行”按钮，即可开始游戏。
